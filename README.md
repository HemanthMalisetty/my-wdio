# my-wdio

My first try on webdriverio.

# Packages used

- `wdio-cli`
- `babel`
- `mocha`
- `chai`
- `allure-commandline`
- `webdriverio`

# Install Packages

```bash
$ npm install
```

# Run Tests

```bash
$ npm test
```

# Generate Allure Report

```bash
$ npm run report:generate
```

# Open Allure Report

```bash
$ npm run report:open
```
