# my-wdio

My first try on webdriverio.

# Packages used

- `wdio-cli`
- `babel`
- `mocha`
- `chai`
- `allure-commandline`
- `webdriverio`
- `wdio-image-comparison-service`
- `fs-extra`

# Install Packages

```bash
$ npm install
```

# Run Tests

```bash
$ npm test
```

# Run Visual Tests with `wdio-image-comparision-service`

```bash
$ npm run test:visual
```

# Generate Allure Report

```bash
$ npm run report:generate
```

# Open Allure Report

```bash
$ npm run report:open
```
